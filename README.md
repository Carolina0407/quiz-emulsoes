
# Quiz sobre Emulsões

Este é um quiz interativo sobre emulsões, ideal para uso em estandes e eventos.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub (ex: `quiz-emulsoes`).
2. Envie o arquivo `quiz_emulsoes_com_nome_ranking.html` para o repositório.
3. Vá em **Settings → Pages**.
4. Em **Source**, selecione a branch principal (`main`) e a pasta raiz (`/`).
5. O GitHub gerará um link público, por exemplo:  
   `https://seuusuario.github.io/quiz-emulsoes/quiz_emulsoes_com_nome_ranking.html`

## QR Code

O QR Code gerado neste pacote aponta para um link fictício. Após publicar no GitHub Pages, atualize o QR Code com o link real.
